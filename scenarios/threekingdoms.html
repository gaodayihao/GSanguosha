三国鼎立

综述
三国鼎立模式中身份用上家下家关系来确认，按逆时针的出牌顺序来区分上下家
行动顺序
3人局随机位置

游戏流程

三名玩家围成一个三角形扮演三个国家，形成三国足鼎立局面。

发牌阶段

1、每位玩家分发四张手牌，起始四张手牌中有武将牌的可以装备武将牌。

 

2、当你装备武将牌后，你就具有了武将的技能，一旦失去武将牌则相应的武将技能也会消失。

 

3、当你没装备武将牌时，你为4血上限的暗将，性别默认为中性。

判定阶段

若判定牌为武将牌，则重新进行一次判定，直果结果不是武将牌为止。任何时候的判定同理。

摸牌-出牌阶段

1、每回合开始时能摸取2张牌，如果摸取的牌中有武将牌，则在本回合内可以将武将牌弃掉再摸取一张牌。摸牌阶段内摸起的武将牌每回合内只能重铸一次。例如：到你的摸牌阶段，摸起两张武将牌，你的本回合内可以将这两张武将牌弃掉重新摸取两张牌，若此时再摸起武将牌，则不能再重铸

 

2、在自己的出牌阶段内可以装备武将牌，若之前装备有武将，新装备上的武将替换掉原来的武将，被替换的武将牌进入弃牌堆。同时你具备有新武将的技能，旧武将的技能消失。每阶段限一次。

 

3、武将牌在武将牌区或者装备中时，除了锦囊牌【过河拆桥】和【顺手牵羊】（顺手牵羊须开启特殊设定）能对武将牌生效，其他一切武将技能或者锦囊牌不能对武将牌生效。

弃牌阶段

1、每回合结束时，手牌上限的计算和普通身份局相同。武将牌区的武将牌除去已装备的武将牌后，不

 

2、每回合结束时，多余的手牌和武将牌需要弃掉。

Note
 本模式默认禁将为：于吉、左慈、SP庞统
玩家可以自行在禁表中修改，但禁表不能为空
另：该模式禁表须重启游戏后生效

胜利条件

当你的下家阵亡，而你存活则你算获胜。
你的上家阵亡或者自己阵亡，则你算失败。

特殊规定

  1. 普通牌的变体：过河拆桥：抽取其他角色的一张牌（手牌或装备区的牌或判定区的牌）后弃掉，或者选择抽取该角色的一张武将牌（可以是已经装备上的武将）后弃掉，同时需要弃掉自己的一张武将牌区内的武将牌（自己已装备的武将牌不计入内）。

  2. 任何指定性别的技能都不能对无性的角色生效。

  3. 角色的体力上限跟装备的武将体力上限保持一致。角色的体力上限变化时只能根据上限减少，不能增加。例如：当你是4体力4上限时装备3体力上限武将，则此时你的上限变为3，同时体力变为3血。当你是3体力3上限时装备4体力上限武将，则此时体力上限变为4，体力为3。

  4. 三国鼎立模式中不能启用双将，不能自由分配身份。

Note
 武将牌区解释：摸起来的武将牌都放在武将牌区，不算在手牌里面。每回合结束时，武将牌区最多只能有两张武将牌。

村规

  1. 顺手牵羊：抽取其他角色的一张牌（手牌或装备区的牌或判定区的牌）后获得，或者选择抽取该角色的一张武将牌（可以是已经装备上的武将）后立即装备上，同时需要替换掉自己已装备的武将牌，当自己没有装备武将牌时不能牵取其他角色的武将牌。

  2. 已装备的武将牌虽然显示在武将牌区，但它并不属于那里，各种结算时不会计算入内。

  3. 当摸完牌堆后游戏仍未结束则平局。

  4. 武将区的牌是公开相互可见的。

Note
 这些都是村规

奖励与惩罚

因当有一个角色阵亡后游戏随之结束，所以并无奖惩。
